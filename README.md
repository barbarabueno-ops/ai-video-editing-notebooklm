# ai-video-editing-notebooklm
Este repositório reúne e organiza ferramentas de Inteligência Artificial voltadas para o dia a dia de editores de vídeo, com foco em produtividade, qualidade e otimização de workflow.  Diferente de listas genéricas, este projeto apresenta uma curadoria prática, mostrando como cada ferramenta pode ser aplicada em situações reais de edição.
